# Embedded Web Server (ESP32)

## Project Overview

This project demonstrates how to host a web server using the ESP32 microcontroller. The ESP32 connects to a Wi-Fi network and serves a web page that can be accessed through a browser.

## Features

- Wi-Fi connectivity
- Embedded web server
- Remote device monitoring
- Browser-based control interface
- Real-time status display

## Components Required

- ESP32 Development Board
- Wi-Fi Network
- USB Cable
- Computer or Mobile Browser

## Technologies Used

- ESP32
- Embedded Systems
- HTML
- C++
- IoT

## Working Principle

1. ESP32 connects to Wi-Fi.
2. Web server starts on ESP32.
3. User enters ESP32 IP address in browser.
4. Web page is displayed.
5. User can monitor or control connected devices.
